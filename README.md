# Alice 3 Project

A fun, interactive game that tests your **<ins>math skills</ins>** to escape the maze and win!

[Download Alice 3](https://www.alice.org/get-alice/alice-3/)

## Requirements
- Alice 3 application
- _Knowledge of PEMDAS, percentages, rounding, measurements, and inequalities_

## Overview
After opening the game, users can begin playing immediately. Users will be answering math related questions to win the game. Win the game by answering three questions correctly. Lose the game when you answer 3 questions incorrectly.

## Instructions
1. Open game
2. Read instructions to play game from alien
3. Read and answer questions
4. Read hints if needed
5. Game automatically restarts when you win or lose
