# Alice 3 Project

A fun, interactive game that tests your **<ins>math skills</ins>** to escape the maze and win!

[Download Alice 3](https://www.alice.org/get-alice/alice-3/)

## Requirements
- Alice 3 application
- _Knowledge of PEMDAS, percentages, rounding, measurements, and inequalities_

## Overview
After opening the game, users can begin playing immediately.

## Instructions
1. Open game
2. Read instructions to play game from alien
3. Read questions, answer, read hints
4. Get 3 answer correct and you win
5. Get 3 answers wrong and you lose
6. Game automatically restarts when you win or lose
